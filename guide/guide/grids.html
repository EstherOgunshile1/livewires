Grids

We’re using a complex folding compound grid for maximum flexibility in our responsive layouts. This is a fluid grid with flexible columns and fixed-width gutters to maintain proportional spacing and alignment for all screen dimensions.

Our grid system is not attached to classes like other pre-packaged CSS grid systems. Instead, they are set in SCSS using our simple grid varables. This keeps our markup clean and our layout much easier to modify at various breakpoints. You can even combine them to make interesting compound grids.

E.G. To specify a width of four columns of a six column grid ($c6 == 1 column of a 6-column grid), use the following:

    /* Create a 4-column object centered in a 6-column grid */
    .object{
      @include gutters; /* Applies gutter spacing to sides of object. */
      width:$c6*4; /* $c6 = 1 column of a 6-column grid. Mulitply by 4 to specify 4 columns wide. */
      margin-left:$c6*1; /* Moves object 1 column to the right. */
    }
